واجهة API لاختصار الروابط

استخدم واجهة API هذه لربط موقعك أو تطبيقك بخدمة اختصار الروابط من elgoharyX

POST إنشاء رابط مختصر
https://earc-eg.github.io/api/shorten
Headers:
Content-Type: application/json
X-API-Key: your-api-key-here
Body (JSON):
{
  "url": "https://example.com/very/long/url",
  "slug": "my-custom-link"  // اختياري
}
الرد (JSON):
{
  "success": true,
  "slug": "my-custom-link",
  "short_url": "https://earc-eg.github.io/my-custom-link",
  "original_url": "https://example.com/very/long/url"
}
🔑 مفاتيح API

يمكنك إدارة عدة مفاتيح، كل مفتاح مرتبط بحسابك وتاريخ إنشائه.

🕘 تاريخ الاستخدام
📘 أمثلة الاستخدام
JavaScript (Fetch)
<script src="./shortener.js"></script>
<script>
  (async () => {
    const result = await window.elgoharyxShorten({
      apiKey: 'YOUR_API_KEY',
      url: 'https://example.com'
    });
    console.log(result.short_url);
  })();
</script>
Python
import requests

response = requests.post(
  'https://earc-eg.github.io/api/shorten',
  headers={'X-API-Key': 'YOUR_API_KEY'},
  json={'url': 'https://example.com'}
)
print(response.json()['short_url'])
PHP (cURL)
$ch = curl_init('https://earc-eg.github.io/api/shorten');
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
  'Content-Type: application/json',
  'X-API-Key: YOUR_API_KEY'
]);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([
  'url' => 'https://example.com'
]));
$response = curl_exec($ch);
$data = json_decode($response, true);
echo $data['short_url'];
